Esta aplicación utiliza en el lenguaje HTML, CSS y JS dichos programas fueron elegidos para el desarrollo y creación de este sitio web.

El lenguaje HTML nos permitió un despliegue rápido, a reconocer y admitir cualquier tipo de explorador y desarrollar archivos pequeños.

En el lenguaje CSS se utilizo como presentación de los documentos HTML. Podemos decir que el lenguaje CSS nos sirvió para la organización y aspectos de la pagina web, nos permitió manejar multitud de opciones de presentación, como ejemplo, colores, tamaño y tipo de letras, etc.

Y finalmente el lenguaje JS donde lo utilizamos para mejorar el interfaz del usuario en nuestra pagina web, para construir la aplicación y automatizar acciones o bien hacerla mas compleja y personalizada.
