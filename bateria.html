Se aplicaron ocho buttons que corresponden cada uno a las partes de una bateria, para poder llevar a cabo este programa creamos; button.RideCymbal, button.FloorTom, button.MidTom, button.BassDrum, button.HighTom, button.SnareDrum, button.CrashCymbal y button.HiHat. También aplicamos el uso de audio, programándolo así para que cada button emita el sonido de audio correspondido a su toque.
